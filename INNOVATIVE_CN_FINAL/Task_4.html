Task 4

Network Packet Routing & Time-Based TTL

This simulation uses Dijkstra's algorithm to find the lowest-cost path in a 40-node network. The packet's Time-To-Live (TTL) is a time budget in seconds (s). At each hop, the link's cost is subtracted from the TTL. The packet is dropped if the TTL is insufficient for the next hop(i.e. packet expiration).
