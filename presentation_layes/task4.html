Header Field Information

The header field is crucial in the Presentation Layer as it tells the receiving end how to properly decrypt, decompress, and decode the data. Standard fields include:

Encryption Type
Compression Type
Encoding Type
Protocol Version
Timestamp
Data Length
Checksum
